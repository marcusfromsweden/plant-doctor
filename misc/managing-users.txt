* Adding a new user
  - To add a new user, call POST on http://localhost:8080/auth/addNewUser with body as follows:
  {
      "name": "marcus",
      "password": "123",
      "email": "marcus@email.com",
      "roles": "ROLE_USER"
  }
* Generate token
POST http://localhost:8080/auth/generateToken
{
    "username": "marcus@email.com",
    "password": "123"
}
